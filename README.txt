RESHMA ARCHITECTURE PORTFOLIO WEBSITE

GITHUB PAGES UPLOAD
1. Upload ALL files and folders from this package to the root of your GitHub repository.
2. The root must contain: index.html, style.css, script.js, assets folder and the PDF.
3. Open GitHub repository > Settings > Pages.
4. Source: Deploy from a branch.
5. Branch: main. Folder: / (root). Click Save.
6. Wait 1-3 minutes and open the published GitHub Pages link.

CONTACT FORM
Open script.js and replace:
const CONTACT_EMAIL = "";
with Reshma's email, for example:
const CONTACT_EMAIL = "name@example.com";

The form uses the visitor's default email application and does not require a server.
